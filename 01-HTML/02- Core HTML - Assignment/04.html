tag lists

  1. title - The title tag in HTML is used to define the title of HTML document It sets the title in the browser toolbar.
  2. p - defines paragraph of text
  3. b - defines bold text
  4. h1- defines biggest heading of tag
  5. a - anchor tag is used to create a hyperlink on the webpage.

  7. dl - The dl tag in HTML is used to represent the description list. In HTML
  8. main - The main tag is used to give the main information of a document.
  9. small - The small tag in HTML is used to set small font sizes.
  10. style - The style tag in HTML helps us to design the web page.
